# SCT_DA_3.
HR EMPLOYEE ATTRITION ANALYSIS DASHBOARD

Transforming HR Data into Actionable Insights with Power BI

PROJECT OVERVIEW

This interactive Power BI dashboard analyzes employee attrition patterns to identify the key factors influencing workforce turnover. The objective was to answer a critical business question:

"Why are employees leaving the company?"

TOOLS & TECHNOLOGIES

• Power BI
• Power Query
• DAX
• Data Modeling
• Data Visualization

KEY DASHBOARD FEATURES

✓ Attrition Rate Analysis
✓ Department-wise Attrition
✓ Age Group Analysis
✓ Job Role Insights
✓ Overtime Impact Assessment
✓ Salary vs Attrition Correlation
✓ Interactive Slicers & Filters

BUSINESS INSIGHTS

• Sales department recorded the highest employee turnover.
• Employees aged 26–35 showed the highest attrition rate.
• Overtime was strongly associated with employee exits.
• Lower-income employees were more likely to leave.
• Employees with shorter tenure demonstrated higher turnover.

BUSINESS IMPACT

The dashboard enables HR teams to:
• Identify high-risk employee groups
• Improve retention strategies
• Support workforce planning
• Make data-driven HR decisions
